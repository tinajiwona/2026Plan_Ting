# 📋 2026 Plan Schedule (2026年计划安排)

A bilingual interactive web-based planner for organizing your 2026 plans with automatic saving functionality.

一个双语交互式网页计划工具，用于组织你的2026年计划，支持自动保存。

## 🌍 Language Support / 语言支持

- 🇨🇳 **Chinese (中文)**: [2026_plan.html](2026_plan.html)
- 🇬🇧 **English**: [2026_plan_en.html](2026_plan_en.html)

## ✨ Features / 功能特点

### General Features / 通用功能
- ✏️ **In-place Editing** / 原地编辑 - Click any cell to edit directly in the browser
- 💾 **Auto-Save** / 自动保存 - Data is automatically saved to browser's localStorage
- 🔄 **Data Sync** / 数据同步 - Changes sync between Chinese and English versions
- ➕ **Add Rows** / 添加行 - Create new entries with the "+ Add Row" button
- 🗑️ **Delete Rows** / 删除行 - Remove unwanted entries easily
- 📱 **Responsive Design** / 响应式设计 - Works on desktop and mobile devices

### Plan Categories / 计划分类

#### 📚 Paper Plans / 论文计划
- **Paper Revision Plan** / 论文修改计划
  - Paper Title / 论文名
  - Status (Review Round 1/2, Round 1/2 Complete) / 审查状态
  - Timeline / 时间

- **Paper Writing Plan** / 论文撰写计划
  - Paper Title / 论文名
  - Key Highlights / 高亮内容
  - Keywords / 关键词
  - Timeline / 时间

#### 🎓 Learning Plan / 学习计划
- Subject / 学习科目
- Goal / 学习目标
- Timeline / 时间安排

#### 💼 Job Offers Plan / 工作Offer计划
- **Industry Opportunities** / 业界机会
  - Company/Position / 公司/职位
  - Job Description / 职位描述
  - Contact/Notes / 联系方式/备注

- **Academic Opportunities** / 学界机会
  - University/Position / 学校/职位
  - Job Description / 职位描述
  - Contact/Notes / 联系方式/备注

## 🚀 Getting Started / 快速开始

1. **Clone the repository** / 克隆仓库
   ```bash
   git clone https://github.com/tinajiwona/2026Plan_Ting.git
   cd 2026Plan_Ting
   ```

2. **Open in browser** / 在浏览器中打开
   - Open `2026_plan.html` for Chinese version / 打开`2026_plan.html`查看中文版本
   - Open `2026_plan_en.html` for English version / 打开`2026_plan_en.html`查看英文版本

3. **Start Planning** / 开始规划
   - Click any cell to edit / 点击任何单元格进行编辑
   - Data saves automatically to your browser / 数据会自动保存到浏览器中

## 💡 Tips / 使用提示

- **Switching Languages** / 切换语言 - Use the language buttons in the header to switch between Chinese and English
- **Data Persistence** / 数据持久化 - Your data is stored in browser localStorage and persists across sessions
- **Keyboard Shortcuts** / 键盘快捷键
  - Click "Save" button or press Enter to confirm edits / 点击保存按钮或按Enter确认编辑
  - Click "Cancel" button or press Escape to discard changes / 点击取消按钮或按Esc放弃编辑

## 🔒 Privacy / 隐私

All data is stored locally in your browser. No server uploads or cloud storage is involved. Your planning information remains completely private on your device.

所有数据都存储在你的浏览器中。不涉及服务器上传或云存储。你的规划信息完全保留在你的设备上。

## 🛠️ Technical Stack / 技术栈

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Storage**: Browser localStorage API
- **No Dependencies**: Pure vanilla web technologies, no frameworks or libraries required
